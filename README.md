# これはコマンドチェック（仮）アプリです
## まずは…
### ディレクトリ内でdocker-compose up -dを実行
### コンテナに入ってgo run main.goを実行して使用します
## できること
### cd,mkdir,ls,pwdの４種類のコマンドを判断し
### 正しいか、間違ってた場合近いものを出力します
### アプリが起動したら入力を求められるため上記４つのコマンドを入力またはそれに近いものを入力してください
### 終了する場合はexitを入力すると終了します
